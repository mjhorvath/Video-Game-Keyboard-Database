Description:

This form generates a printable keyboard diagram in a new window, depicting keyboard bindings for various video games and other software.

Instructions:

  1. Select a scheme (key text & captions), style (visual formatting), effect (even more visual formatting), and layout (key positions) using the forms, above. Once you've made your selection, click on the 'Create Diagram' button. A new window will appear.[2][3]
  2. Print the page in the new window at 96dpi (dots-per-inch).[4][5]
  3. Optionally, cut along the dotted lines and lay over the top of your keyboard.

Submissions:

Fill out this online form to submit a scheme for a new game. Note that you will have to abide by the LGPL license as per the project as a whole.

Alternately, you can view and edit the following sample files: schemes (here and here), layouts (here and here), styles (here and here). To submit the modified file, contact me via this form, and I will reply to you with further instructions.

Footnotes:

  1. Microsoft Internet Explorer is required for 'Effects'.
  2. JavaScript must be enabled in the browser. If JavaScript is not enabled, then the selection boxes above should appear empty.
  3. Not all layouts are currently compatible with all games (schemes). In Mozilla Firefox, incompatible layouts should automatically be disabled.
  4. Printing doesn't work properly, at the moment. You could try capturing your screen at high resolution and using a paint program to print the image. Or, if you're browsing using Internet Explorer, you can select the 'Rotate 90°' effect to display the chart in a vertical orientation (which will print properly). Newer browsers have an 'shrink to fit' option when printing and a 'zoom' feature when browsing, but these don't always help. I am interested in hearing feedback from those who have experience with printing to adhesive paper.
  5. Remember to enable the printing of background colors and images in 'Internet Options' (Internet Explorer) or 'Page Setup' (Firefox) when printing.
